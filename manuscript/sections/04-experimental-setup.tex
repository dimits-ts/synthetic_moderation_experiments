\subsection{Synthetic Discussion Generation}
\label{ssec:appendix:discussion}

We use a set of starting (``seed'') discussions taken from random points in escalated discussions featured in the CMV-Awry dataset \citep{chang-danescu-niculescu-mizil-2019-trouble} (see Appendix~\ref{TODO} for details). We then run $N_d=20$ discussions for each pair of facilitation strategies $S$ and LLM (\S\ref{ssec:experimental:setup}) with each discussion running for up to $30$ comments (including facilitator interventions). An overview of how the experiments are generated can be found in Alg.~\ref{alg:exp_generation}. The \textit{Sample} function returns a number of samples from a set following the uniform distribution, while the \textit{Actors} function creates an LLM agent using a model and a prompt.

\begin{algorithm}[t]
	\caption{\ac{SDG} setup}
	\label{alg:exp_generation}
	\hspace*{\algorithmicindent} \textbf{Input:} 
	\begin{itemize}[noitemsep, nosep]
		\item User SDBs $\Theta = \{\theta_1, \dots, \theta_{45}\}$
		\item Strategies $S = \{s_1, \ldots, s_4\}$
		\item Seed discussions $O = \{o_1, \ldots\}$
		\item LLMs = $\{LLaMa, Mistral, Qwen, OLMo\}$
	\end{itemize}
	\hspace*{\algorithmicindent} \textbf{Output:} Set of discussions $D$
	\begin{algorithmic}[1]
		\State $D = \{\}$
		\For{$llm \in LLMs$}
		\For{$s \in S$}
		\For{$i = 1, 2, \ldots, N_d$}
		\State $\hat{\Theta} = $ \Call{Sample}{$\Theta, num=7$}
		\State $U =$  \Call{Actors}{llm, $\hat{\Theta}$}
		\State $m = $ \Call{Actors}{llm, $s$}
		\State $o = $ \Call{Sample}{$O, num=1$}
		\State $d =$ \{users: $U$, mod: $m$, topic: $o$\}
		\State $D = D \cup d$
		\EndFor
		\EndFor
		\EndFor
		\State \Return $D$
	\end{algorithmic}
\end{algorithm}


\subsection{Facilitation Strategies}
\label{ssec:experimental:strategies}

We test two facilitation strategies used in real-life and compare them with two baselines. Note that the process of turning sometimes extensive documents into short prompts, necessitated by open-source LLMs, is necessarily imperfect. We leave the optimal derivation of strategy prompts to future work.

\begin{enumerate}[nosep, noitemsep]
    \item \textbf{\strategynomod}: A \emph{common} strategy where no facilitator is present.

    \item \textbf{\strategynoinstr}: A \emph{common} strategy where a LLM facilitator is present, but is provided only with basic instructions. This approach is already being used in some platforms \citep{Tsai_Deliberate_Lab_Open-Source_2025}. Example: “You are a moderator, keep the discussion civil”. 

    \item \textbf{\strategyregroom}: A \emph{real-life} strategy based on guidelines given to human facilitators of the ``Regulation Room'' platform \citep{Cornell_eRulemaking2017}. The instructions are suitable for online fora, where facilitators also engage in content moderation, and their effectiveness must be balanced by their throughput. Example: ``Stick to a maximum of two questions, use simple and clear language, deal with off-topic comments''.

    \item \textbf{\strategyconstrcomm}: A \emph{real-life} strategy based on the human facilitation guidelines \citep{dimitra-book}. It approaches facilitation from a more personalized and indirect angle, forbidding facilitators from directly providing opinions or directions. This makes the strategy ideal for deliberative environments. Example: ``Do not make decisions, be a guide, provide explanations''.
\end{enumerate}

In order to gauge the quality of our synthetic discussions, since we can not reliably measure ``realism'' (\S\ref{ssec:related:human-llm}), we use the ``diversity'' metric \citep{ulmer2024}. Low diversity points to pathological problems (e.g., LLMs repeating previous comments). Conversely, extremely high diversity may indicate a lack of interaction between participants. A discussion where participants engage with each other will naturally feature some lexical overlap, including common terms or paraphrasing points made by others. We compare the distribution of diversity scores for synthetic discussions with that measured on sampled human discussions. This allows us to estimate the extent to which synthetic discussions approximate real-world content variety and participant interaction. 


\subsection{Technical Details}
\label{ssec:experimental:setup}

 We use four instruction-tuned, open-source models: LLaMa3.3 (70B), Qwen2.5 (33B),  Mistral Nemo (12B) and OLMo2 (32B), quantized to 4 bits. All the experiments were collectively completed within four weeks of computational time, using two Quadro RTX 6000 GPUs. The execution script is available in the project's repository.
