\begin{abstract}
	Human facilitation is insufficiently scalable for modern online spaces. While Large Language Models (LLMs) may be able to automate the facilitation of online discussions, no large-scale studies have tested this hypothesis due to significant costs associated with human involvement. An effective solution is synthetic discussion simulations to create initial pilot experiments. We establish design principles and create a new Synthetic Discussion Generation (SDG) framework, which we use to prototype the development of LLM facilitators by generating data without human involvement. Through fast and cheap experiments, we find severe limitations on the capabilities of LLM facilitators; they cannot utilize effectively human facilitation strategies, nor decide when to intervene in a discussion. We develop an open-source python framework  and release a large, publicly available dataset containing LLM-generated discussions using multiple open-source models. These resources can be used for LLM facilitator finetuning as well as behavioral analysis of current out-of-the-box LLMs in the task.
\end{abstract}
